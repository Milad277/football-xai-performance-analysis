# Notebooks

This folder contains the Jupyter notebooks used in the football XAI performance analysis project.

## Notebook Description

### 01_data_exploration.ipynb

Initial data exploration notebook.  
This notebook is used to inspect the structure of the football event data, understand the available columns, and explore the raw Wyscout data before feature engineering.

### 02_per_match_feature_engineering.ipynb

Feature engineering notebook for creating match-level and team-level features.  
This includes aggregating football events, preparing home and away team statistics, and generating features used for machine learning.

### 03_rulex_modeling_workflow.ipynb

Notebook related to the Rulex modeling workflow.  
This notebook documents the preparation of data for explainable machine learning and the analysis of model outputs from the Rulex platform.

### 04_results_analysis.ipynb

Results analysis notebook.  
This notebook is used to review model performance, compare outputs, and interpret the most important tactical features.

## Notes

The notebooks are part of an academic project on Explainable Artificial Intelligence in football analytics.  
The main objective is to transform football event data into interpretable tactical features and use them for explainable machine learning analysis.
